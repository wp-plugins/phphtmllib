Class RemoteCSVFILEDataListSource

Description
Description | Vars | Methods (details)

This DataListSource child class gets the data from a CSV (comma seperated values) file on disk on a remote computer.

The CSV MUST have a 'header' line before any data. The 'header' line is a CSV line that provides a name for each column of data in the file.

All lines that start with '#' or '//' are ignored as comments.
