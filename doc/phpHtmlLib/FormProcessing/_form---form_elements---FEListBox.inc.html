Class Description
FEListBox This is the ListBox FormElement which builds a select field with all of its options.
FEYesNoListBox Build a Yes/No Select box
FENestedListBox This class builds a nested select box that is used to select an entry from nested levels.
FEMultiListBox This is the MultiListBox FormElement which builds a select field with all of its options. It enables the ability to have multiple selections.
FEUnitedStates This class builds a FEDataList that shows a select box for States of the U.S.A.
FEEuropeanUnion This class builds a FEDataList that shows a select box for states of the European Union.
FEComboListBox This builds a complex dual select box with buttons to move entries from one select box to another.
FEComboListButtonsBox This class builds the FEComboList with buttons below each element
FEComboListBoxVertical This builds a complex dual select box with buttons to move entries from one select box to another. The boxes are stacked vertically instead of horizontally.
FEMonths This class builds a FEDataList that shows a select box for Months of the year
FEYears This class builds a FEDataList that shows a select box listing a range of years
FEDays This class builds a FEDataList that shows a select box listing the days of the month
FEDate This class builds a widget that shows a group of select boxes (FEYears, FEMonths, FEDays) representing a date.
FEDay This class builds a widget that shows a group of select boxes (FEMonths, FEDays) representing a date.
FEHoursListBox this class is used for building a listbox for displaying Hours.
FEMinutesListBox this class is used for building a listbox for displaying Minutes.
FESecondsListBox this class is used for building a listbox for displaying Seconds.
FEGMTTimeZoneListBox This class is used to build a ListBox for GMT timezone selection
FEWeekdaysListBox This Form Element builds a list box of the days of the week.
