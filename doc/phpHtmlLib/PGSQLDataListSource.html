Class PGSQLDataListSource

Description
Description | Vars (details) | Methods (details)

This SQLDataListSource child class interacts with

with a PostgreSQL Database using built in pg_* calls. This assumes that you have PostgreSQL support compiled into PHP

How to use? in the DataList child's get_data_source() method you pass in the result of the pg_connect() call to the constructor.
